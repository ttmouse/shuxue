# 工作边界

## 项目范围

`数学口算大闯关` 是一个零依赖的纯前端小学口算练习应用。

## 允许的操作

- 修改 HTML 结构（`index.html`）
- 修改 CSS 样式（`css/style.css`）
- 修改 JS 逻辑（`js/*.js`）
- 添加新题型生成器
- 优化 UI/UX

## 禁止/限制的操作

| 操作 | 原因 |
|---|---|
| **引入 npm 包** | 项目零依赖，引入包会破坏离线可用性 |
| **引入构建工具** | 无构建流程，纯静态文件直接加载 |
| **切换到 SPA 框架** | 项目规模无需框架，增加复杂度 |
| **添加后端依赖** | 数据全部存在 localStorage，不需要后端 |
| **改变存储方式** | localStorage 是故意选择（离线可用、零运维） |
| **添加登录/用户系统** | 超出项目范围，目标群体是本地使用 |

## 已知风险

- **无测试覆盖** — 手动测试为主，修改后需手动验证所有页面流程
- **localStorage 容量限制** — 约 5MB，长时间使用错题本过多可能溢出
- **无数据导出** — 错题和统计仅限本地浏览器，切换设备丢失
- **emoji 作为 UI 元素** — 首页 logo(`🧮`)、结果页图标(`🏆`)、错题本日历(`📅`) 等使用了 emoji。如果需要支持主题定制或跨平台一致渲染，应替换为 SVG

## 设计约束

- 保持「可心算」原则 — 所有题目无需纸笔
- CSS 变量系统 — 颜色/圆角/阴影通过 `:root` 变量控制
- 响应式 — 支持移动端（`@media (max-width: 480px)`）
- 知识点按年级分组 — 1~2 年级 / 3~4 年级 / 5~6 年级
- 难度三档 — easy / medium / hard
